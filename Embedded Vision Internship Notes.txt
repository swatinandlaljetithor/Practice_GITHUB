			# Topic : Message QUEUE

- message Queue is linear type of data structure used to store data in a sequentially.
- it save data in FIFO manner.
- while removing data it remove recently added data first.
 
- Advantages of Queue
	Data queues are fast and optimized.
	Queues are flexible.
	They can handle multiple data types. 

- Enqueue:
	- it is used to add item to the queue.
	- if queue is full then it is said to be an overflow condition.
	- Time complexity : O(1)

- Deque :
	- it is used to remove an item from the queue.
	- it item are popped in the same order in which they are pushed.
	- if queue is empty - Underflow condition.
	- Time Complexity : O(1)

- Implimentation of Queues:
	1) list
	2) collections.deque
	3) queue.Queue
